# Bitcoin Utils in Crystal

## Genesis

This started with a port of [James D'Angelo's Python scripts](https://github.com/wobine/blackboard101) to [crystal-lang](https://crystal-lang.org).

Functionality:
 - generate public key
 - sign data
 - verify data
